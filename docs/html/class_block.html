Detailed Description

Used to represent collidable game blocks.

Available collidable Block types:

  • BlockRed,
  • BlockBlue,
  • BlockBrown,
  • BlockGray,
  • BlockGreen,
  • BlockPurple,
  • BlockYellow,
  • BlockCrate,
  • Water1,
  • Water2,
  • Sign1,
  • Sign2,
See also
Obj::Entity
Note
One specific texture e.g. BlockYellow is loaded only once to the memory.
