Detailed Description

Encapsulates the logic of paused menu within one StatePaused class.

Note
All possible transitions from StateMenu are:
  • StateGame
  • StateOptions
  • StateMenu

Inherits from State base. Created inside Game and Stored inside StateMachine. Tbe ID of StateMenu can be retrieved from state::menuID.
