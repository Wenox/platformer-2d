Detailed Description

Encapsulates the logic of map loader menu within one StateMapLoader class.

Note
All possible transitions from StateMapLoader are:
  • StateGame
  • StateMenu

Inherits from State base. Created inside Game and Stored inside StateMachine. Tbe ID of StateMenu can be retrieved from state::menuID.
